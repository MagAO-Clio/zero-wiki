Above: Clio camera control GUI. This is what you interact with to take data.

Reset==reset electronics on the detector. Do this if you are unable to take any images and are repeatedly getting errors. The yellow bar above will be red. After hitting reset, click Init to initialize the program. If that fails again, click Reset again. Repeat as many times as it fails (it's finicky, eventually it should work).

Format==choose your image size

Take Back==take and store image to use for background subtraction in video mode

Video==take continuous images and display them on ds9

Use Back==check if you wish to subtract the background image (obtained by clicking Take Back) from subsequent displayed images

Save Data in Cubes==check this if you want 3D data, rather than 2D (coadded) data. NOTE that cube data files are much larger than coadded data.

Tel. info in header==this should always be checked, toggles whether header information from the telescope is populated in saved Clio images

Take Cal frame==check this if you wish to automatically take minimum exposure images at every nod position. This is useful for obtaining unsaturated images of the star for photometry.

Set Directory==you shouldn't have to mess with this, as directories for saving data are created automatically. Allows you to choose where to save your data.

Set Prefix==what your data files will be called, e.g., "YourFavoriteStar00001.fit" would be saved if you specified "YourFavoriteStar" as the prefix (without quotes).

Last image number==you shouldn't have to mess with this, as the next image to be taken is automatically updated, but this allows you to manually specify what the number of the next file should be.

xnod/ynod==set the nod vector here

Integration==set integration time for image acquisition in mas

Coadds==self-explanatory

Images per nod==set the number of images to take at each nod position

Nod cycles==set how many times you want to repeat a given nod cycle of image acquisition

Quick Video==minimum exposure, 1-coadd video mode. Good for aligning

Take Images==take data without nodding

Take Images (Nod AB)==take images at position A, move the telescope to position B, take data, stop.

Take Images (Nod ABBA)==take images at position A, move the telescope to position B, take images, take images, move the telescope to position A, take images, stop.

Script==choose script to execute for observations. More on this at the bottom of the page.
